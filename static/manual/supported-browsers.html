brat browser support

brat is implemented using standard web technologies and should, in principle, be usable in any standards-compliant browser.

In practice not all browsers currently fully support all aspects of the technologies used to implement visualization and editing in brat. The following table summarizes the current browser support for various brat functions.

Browser Visualization Editing
Chrome (Google) on PC Full Full
Chrome (Google) on Android (tablet/phone) Full TODO
Firefox (Mozilla) Full Partial
Opera Full TODO
Safari (Apple) on PC Full TODO
Safari (Apple) on iPad/iPhone Full TODO
Internet Explorer (Microsoft) version 9 Full Partial
Internet Explorer (Microsoft) version < 9 No No

Please note the following:

  • Partial means ... TODO
  • The above table is based on testing with recent browser versions. Older versions may not support the same features.

Note that even if a browser is not fully supported, it may still be possible to use it with brat for visualization or basic editing.

Why doesn't it work in my browser?

The core of brat visualization is based on Scalable Vector Graphics (SVG). Although SVG has been a recommended web standard for over a decade, not all browsers have implemented it; in particular, Internet Explorer has only started supporting SVG from version 9.
